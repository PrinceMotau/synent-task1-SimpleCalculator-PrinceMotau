# synent-task1-SimpleCalculator-Prince

A simple command lien calculator built using Python which allows user to do basic calculations.

# Features
- Input validation
- Take user input
- Arithmetic operations (+,-,*,/)
- Zero division protection.
- Continuous loop until exit.

# How to run
- clone repository (https://github.com/PrinceMotau/calculator_cli.git)
- Move into project folder (cd calculator_cli)
- Run Program
